1.20.1 数据库基本操作速查表

操作 说明 示例 备注 是否PhalApi新增
fetch() 循环获取每一行 while($row = $user->fetch()) { //... } 否
fetchOne() 只获取第一行 $row = $user->where('id', 1)->fetchOne(); 等效于fetchRow() 是
fetchRow() 只获取第一行 $row = $user->where('id', 1)->fetchRow(); 等效于fetchOne() 是
fetchAll() 获取全部的行 $rows = $user->where('id', array(1, 2, 3))->fetchAll(); 等效于fetchRows() 是
fetchRows() 获取全部的行 $rows = $user->where('id', array(1, 2, 3))->fetchRows(); 等效于fetchAll() 是
queryAll($) 复杂查询下获取全部的行 $rows = $user->queryAll($sql, $parmas); 等效于queryRows() 是
queryRows() 复杂查询下获取全部的行 $rows = $user->queryRows($sql, $parmas); 等效于queryAll() 是
insert() 插入数据 $user->insert($data); 原生态操作需要再调用insert_id()获取插入的ID 否
insert_multi() 批量插入 $user->insert_multi($rows); 否
update() 更新数据 $user->where('id', 1)->update($data); 否
insert_update() 插入/更新 不存时插入，存在时更新 否
delete() 删除 $user->where('id', 1)->delete(); 否

温馨提示：
上表中的$user变量为NotORM表实例，可以通过两种方式获得：
1、使用原生态的notorm，即：$user = DI()->notrom->user
2、使用PhalApi_Model_NotORM基类的方式，即：$user = $this->getORM()
